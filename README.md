# algorithm
sorting algorithm and search algorithm  
包含 二分查找，选择排序，插入排序，希尔排序。  

## 编译
* Windows下编译
  * 用g++编译  
   mkdir build  
   cd build  
   cmake -G"Unix Makefiles" ..  
   make  
  * 用ms的cl.exe编译  
   mkdir build  
   cd build  
   cmake ..  
   用vs 2017 / 或者其他vs版本打开 xxx.sln  
   
* linux下编译  
  * mkdir build  
    cd build  
    cmake ..  
    make
